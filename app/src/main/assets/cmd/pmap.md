pmap
===

报告进程的内存映射关系

## 补充说明

**pmap命令** 用于报告进程的内存映射关系，是Linux调试及运维一个很好的工具。

### 语法  

```
pmap(选项)(参数)
```

### 选项  

```
-x：显示扩展格式；
-d：显示设备格式；
-q：不显示头尾行；
-V：显示指定版本。
```

### 参数  

进程号：指定需要显示内存映射关系的进程号，可以是多个进程号。
