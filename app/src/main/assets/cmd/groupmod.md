groupmod
===

更改群组识别码或名称

## 补充说明

**groupmod命令** 更改群组识别码或名称。需要更改群组的识别码或名称时，可用groupmod指令来完成这项工作。

### 语法  

```
groupmod(选项)(参数)
```

### 选项  

```
-g<群组识别码>：设置欲使用的群组识别码；
-o：重复使用群组识别码；
-n<新群组名称>：设置欲使用的群组名称。
```

### 参数  

组名：指定要修改的工作的组名。
