command
===

调用并执行指定的命令

## 补充说明

**command命令** 调用指定的指令并执行，命令执行时不查询shell函数。command命令只能够执行shell内部的命令。

### 语法  

```
command(参数)
```

### 参数  

指令：需要调用的指令及参数。

### 实例  

使用command命令调用执行`echo Linux`，输入如下命令：

```
command echo Linux            #调用执行shell内部指令
```

上面的命令执行后，将调用执行命令`echo Linux`，其执行结果如下：

```
Linux
```
