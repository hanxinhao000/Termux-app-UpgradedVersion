xlsclients
===

列出显示器中的客户端应用程序

## 补充说明

**xlsclients命令** 用来列出显示器中的客户端应用程序。

### 语法  

```
xlsclients(选项)
```

### 选项  

```
-a：列出所有显示器的客户端应用程序信息；
-display<显示器编号>：指定X Server连接的显示器编号，该编号由"0"开始计算，依序递增；
-l：使用详细格式列表；
-m<最大指令长度>：设置显示指令信息的最大长度，单位以字符计算。
```
