mailstat
===

显示到达的邮件状态

## 补充说明

**mailstat命令** 用来显示到达的邮件状态。

### 语法  

```
mailstat(选项)(参数)
```

### 选项  

```
-k：保持邮件日志文件的完整性，不清空日志文件；
-l：使用长格式显示邮件状态；
-m：合并任何错误信息到一行中显示；
-o：使用老的邮件日志邮件；
-t：使用简洁的格式显示邮件状态；
-s：如果没有邮件则不输出任何信息。
```

### 参数  

邮件日志文件：指定要读取邮件日志文件。
